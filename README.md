# Hello World in Tcl

A minimal Tcl implementation of "Hello, World!".

## Installation

### macOS
```bash
brew install tcl-tk
```

### Linux
```bash
sudo apt install tcl  # Ubuntu/Debian
sudo dnf install tcl  # Fedora/RHEL
```

### Windows
Download from https://www.tcl.tk/software/tcltk/

## Running

```bash
tclsh hello.tcl
```

## Code Explanation

Uses puts command for console output.
